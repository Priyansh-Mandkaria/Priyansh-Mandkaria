# 💫 About Me:
👨‍💻 About Me<br>I’m Priyansh Mandkaria, a Computer Science Engineering student passionate about building reliable backend systems and solving complex engineering problems. My core focus lies in backend development, where I design scalable APIs, optimize databases, and build production-oriented systems using Django, REST APIs, and SQL-based architectures.<br>I enjoy working on real-world problem statements — from rule-based risk scoring engines and healthcare systems to ML-driven projects like Emergency Vehicle Detection. I approach development with a strong problem-solving mindset shaped by competitive programming and chess, emphasizing logic, efficiency, and long-term scalability.<br>Beyond coding, I actively contribute to the tech community through hackathons and technical leadership, collaborating with teams to transform ideas into deployable products. Currently, I’m focused on strengthening DSA, system design, and cloud-scale architecture, aiming to grow into a strong Software Development Engineer capable of building high-impact systems.<br>🚀 Always interested in challenging backend problems, impactful collaborations, and engineering discussions.
